Debian

O sistema Operacional Livre

O Debian é um grupo mundial de voluntários que se esforçaram para produzir uma distribuição de sistema operatativo que é composta inteiramente por Software Livre.

Como surgiu o projeto

O projeto Debian foi fundado inicialmente por Ian Murdock em 16 de Agosto, 1993. Neste tempo, o termo distribuição ainda era novo. Ian pretendia que o desenvolvimento fosse totalmente livre. O projeto foi pratocinado pelo FSF e GNU.
